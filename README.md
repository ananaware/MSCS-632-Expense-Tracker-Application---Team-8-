# MSCS 632 — Expense Tracker (Python & C++)
Two-language CLI app to add/list/filter expenses and compute summaries.

## Structure
- `python/` — dict-based storage, datetime, csv
- `cpp/` — struct/class + STL, fstreams
- `docs/` — design & usage
- `sample_data/` — demo CSV
